Todo.py는 직접 코딩으로 작성한 기능으로,  할일 추가, 보기, 수정, 삭제, 종료가 가능합니다. 
할일 추가시 1번을 추가로 누르면 별표 모양의 중요 표시가 함께 적용되어 보기를 눌렀을때 중요 일정을 눈에 띄도록 만들었습니다.
사용법은 간단하게 원하는 기능을 "1. 추가 / 2. 보기 / 3. 수정 / 4. 삭제 / 5. 종료" 중에 선택하시면 됩니다.

다음으로 app.py는 ai 도움을 받아 firebase를 통해 데이터 저장소를 사용하고, streamlit을 통해 UI를 만들었습니다. 
직접 사용 목적으로 영국 BBC 뉴스를 활용한 영어 단어 연습 웹앱입니다.
매일 최신화되는 BBC의 전 분야 기사 중 랜덤으로 문장을 선택하고, 단어를 빈칸 처리해서 맞출 시 점수가 올라가고, 닉네임에 따른 점수 데이터가 firebase 데이터 저장소에 저장됩니다.
1위부터 5위까지 순위표를 만들어 재미 요소를 붙였습니다. (순위표는 매일 갱신되도록 하였습니다.)
실제 어플 제작까지 가보고 싶었지만 다소 어려움이 있어 웹으로만 만들었습니다.
사용방식은 닉네임 입력 후 나오는 문제들을 맞추면 되고, 3회 안에 못맞출 시 점수 없이 다음 문제로 넘어가게 됩니다.
힌트 사용시 해당 단어의 뜻과 첫 알파벳을 알 수 있습니다.
